# /bluespec.list

📒 List every finding **Blue Spec** is tracking, by name.

```bash
/bluespec.list
```

`/bluespec.list` is a read-only convenience command that prints the findings currently tracked, by name. It depends on no phase, changes nothing, and is the one direct window onto what the tracking map holds. It only reads and reports.
